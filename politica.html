Política de Privacidade

Entenda como o portal da 2ª Companhia trata informações preenchidas nas áreas públicas e operacionais.

1. Finalidade do portal

Este portal é utilizado para organização institucional, consultas públicas, registros operacionais, concursos, boletins, prisões, barcas e acesso ao painel privado.


2. Informações coletadas

O site pode solicitar informações como nome, ID/registro, Discord, motivo de ocorrência, dados de veículo, placa, descrição de B.O., dados de prisão e responsável pelo registro.


3. Uso das informações

As informações são usadas para organizar registros internos, exibir consultas públicas, controlar barcas em serviço, vincular boletins a prisões e manter histórico operacional.


4. Login com Discord

O login com Discord é usado para verificar se o usuário possui cargo autorizado. Apenas membros autorizados conseguem acessar o Painel Operacional para criar B.O., registrar prisões ou assumir barcas.


5. Dados públicos

Boletins, prisões e barcas podem ser exibidos em páginas públicas de consulta. Evite inserir informações sensíveis, pessoais ou desnecessárias nos registros.


6. Armazenamento

Na versão atual, algumas informações podem ser salvas no navegador do usuário ou enviadas para um backend, caso a integração seja ativada. Para armazenamento permanente e compartilhado, é necessário usar banco de dados no backend.


7. Segurança

O acesso ao painel deve ser protegido por autenticação Discord e verificação de cargo. Tokens, secrets e credenciais não devem ser publicados no GitHub nem compartilhados.


8. Alterações nesta política

Esta política pode ser atualizada conforme novas funções forem adicionadas ao portal.


9. Contato

Em caso de dúvidas, procure o comando da 2ª Companhia ou use a página de Ouvidoria.
